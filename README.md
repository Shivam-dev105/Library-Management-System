# 📚 Library Management System

A complete **Library Management System (LMS)** designed to automate and manage library operations efficiently. This system replaces traditional manual registers and provides a digital solution for managing books, students, transactions, and fines.

---

## 🚀 Features

### 👨‍💼 Admin (Librarian Panel)
- Student Management (Single & Bulk Registration)
- QR Code Generation (Student & Book)
- Book Management (Add, Edit, Delete, Copies Tracking)
- Issue & Return Books (QR Scan Supported)
- Overdue Management
- Fine Management (Paid / Unpaid Tracking)
- Borrow History
- Academic Setup (Session, Department, Category, Rack)
- Dashboard Analytics
- System Settings (College Info, Fine Settings, Social Links)
- Secure Logout with Confirmation Modal

---

### 👨‍🎓 User (Student Panel)
- Dashboard (Holding Books, Overdue, Pending Fines)
- Search Books (Title, Author, ISBN)
- View Book Availability & Details
- My Issued Books
- Borrow History
- Pay Pending Fines
- Profile Management
- Change Password
- Logout with Confirmation

---

## 📊 Dashboard Highlights
- Books Issued Today
- Books Returned Today
- Fine Payers Today
- Total Fine Collected
- Session-wise Student Data (Active/Inactive)
